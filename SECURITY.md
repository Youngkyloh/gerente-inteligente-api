# 🛡️ Política de Segurança (Security Policy)

A segurança dos dados das lojas e filiais que utilizam o **Gerente Inteligente** é a nossa maior prioridade. Este documento descreve as versões suportadas e como reportar vulnerabilidades de forma segura.

## Versões Suportadas

Atualmente, apenas a versão mais recente em desenvolvimento (Main Branch) recebe atualizações e patches de segurança.

| Versão | Suportada          |
| -------| ------------------ |
| v1.0.x | :white_check_mark: |
| < v1.0 | :x:                |

## Como reportar uma vulnerabilidade

Se você descobrir uma falha de segurança neste repositório, por favor, **não a divulgue publicamente**. O vazamento público pode colocar os dados de fluxo de caixa e estoque de múltiplos lojistas em risco.

**Para reportar a falha:**
1. Utilize a ferramenta **Private vulnerability reporting** na aba "Security" deste repositório no GitHub para nos enviar um aviso confidencial.
2. Forneça o máximo de detalhes possível, incluindo os passos para reproduzir a falha no Streamlit ou na injeção do banco SQLite.

Você receberá um retorno em até 48 horas confirmando o recebimento do relatório. Nós tratamos todas as denúncias com extrema seriedade e aplicaremos um patch de correção o mais rápido possível.
